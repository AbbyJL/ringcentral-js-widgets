## Widget templates
#### Auth-panel
Used for login, can either login RingCentral WebRTC phone or other third party SDK. Basic support includes:
1. Countries selector
2. Remember me (`localStorage`)

- Actions
  - `login`
  - `showCountry(event)`
  - `switchCountry(event)`
- Handlers
- props
  `dom` `username` `extension` `password` `country`
